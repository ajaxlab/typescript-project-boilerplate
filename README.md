# Typescript Project Boilerplate

A boilerplate for a TypeScript project

## Module

* Build

  `npm run build`

* Watch

  * CommonJs

    `npm run dev:cjs`

  * ES2015

    `npm run dev:esm`

## Test

* Mocha & Chai

  `npm run test`

  `npm run test:dev` (watch mode)

* NYC

  `npm run coverage`

## Clean Code

* TS Lint

  `npm run lint`

## Debugger

* VS Code
